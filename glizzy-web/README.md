# Glizzy Web

The browser version of the glizzy machine.

## What it does
- One **GLIZZY** button
- Plays `assets/glizzy.mp3`
- Shows a random hot dog image from the `assets/` folder
- Works on desktop and mobile browsers
- Can be added to the home screen as a simple PWA

## Files
- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `sw.js`
- `assets/glizzy.mp3`
- `assets/hotdog_1.png` through `assets/hotdog_6.png`

## Easiest ways to host it
- GitHub Pages
- Netlify
- Vercel
- Any static web host

## Test locally
If you have Python:

```bash
cd glizzy-web
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
